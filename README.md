
# Discord obfuscator bot based on the ZeroSec-API




## Installation

1. Download the project

2. Run these commands:
```bash
  npm install
```
3. Set your apikey in the ```index.js``` and change all necessary.
4. Start the bot with
   ```bash
  node index.js```

6. Use this command to obfuscate:
```bash
 /obfuscate
```
